# `@elsikora/typescript/no-unnecessary-condition`

<Callout type="info" emoji="🧭">
	**Disabled.** ElsiKora includes this rule as an intentional disabled override in at least one preset scenario.
</Callout>

## Rule Overview

Disallow conditionals where the type is always truthy or always falsy.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.

## Rule Metadata

| Property           | Value        |

| Type               | `suggestion` |
| Fixable            | ❌ No        |
| Has suggestions    | ✅ Yes       |
| Deprecated         | ❌ No        |
| ESLint recommended | `strict`     |
