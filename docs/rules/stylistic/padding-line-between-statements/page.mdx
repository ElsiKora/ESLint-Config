# `@elsikora/stylistic/padding-line-between-statements`

<Callout type="error" emoji="🚨">
	**Error.** ElsiKora reports this rule as an error in at least one preset scenario.
</Callout>

## Rule Overview

Require or disallow padding lines between statements.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.

## Rule Metadata

| Property           | Value     |

| Fixable            | ✅ Yes    |
| Has suggestions    | ❌ No     |
| Deprecated         | ❌ No     |
| ESLint recommended | `unknown` |
