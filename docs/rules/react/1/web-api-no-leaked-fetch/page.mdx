# `@elsikora/react/1/web-api-no-leaked-fetch`

<Callout type="warning" emoji="⚠️">
	**Warning.** ElsiKora reports this rule as a warning in at least one preset scenario.
</Callout>

## Rule Overview

Enforces that every 'fetch' in a component or custom hook has a corresponding 'AbortController' abort in the cleanup function.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.

## Examples

```tsx filename="bad-example.tsx" copy
import { useEffect } from "react";

export function WindowSize() {
	useEffect(() => {
		window.addEventListener("resize", reportSize);
	}, []);

	return <p>Resize the window</p>;
}
```

  </Tabs.Tab>
  <Tabs.Tab>

```tsx filename="good-example.tsx" copy
import { useEffect } from "react";

export function WindowSize() {
	useEffect(() => {
		window.addEventListener("resize", reportSize);

		return () => window.removeEventListener("resize", reportSize);
	}, []);

	return <p>Resize the window</p>;
}
```
