# `@elsikora/typescript/no-unnecessary-type-assertion`

<Callout type="error" emoji="🚨">
	**Error.** ElsiKora reports this rule as an error in at least one preset scenario.
</Callout>

## Rule Overview

Disallow type assertions that do not change the type of an expression.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.
