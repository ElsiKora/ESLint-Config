# `@elsikora/jsdoc/match-name`

<Callout type="info" emoji="🧭">
	**Disabled.** ElsiKora includes this rule as an intentional disabled override in at least one preset scenario.
</Callout>

## Rule Overview

Reports the name portion of a JSDoc tag if matching or not matching a given regular expression.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.
