# `@elsikora/jsdoc/require-yields`

<Callout type="warning" emoji="⚠️">
	**Warning.** ElsiKora reports this rule as a warning in at least one preset scenario.
</Callout>

## Rule Overview

Requires yields are documented with `@yields` tags.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.

/**
 * Iterates enabled users.
 * @param {Array<{ enabled: boolean }>} users - Users to inspect.
 * @yields {{ enabled: boolean }} Enabled user.
 */
function* enabledUsers(users) {
	yield* users.filter((user) => user.enabled);
}
