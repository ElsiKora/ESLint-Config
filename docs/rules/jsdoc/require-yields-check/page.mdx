# `@elsikora/jsdoc/require-yields-check`

<Callout type="warning" emoji="⚠️">
	**Warning.** ElsiKora reports this rule as a warning in at least one preset scenario.
</Callout>

## Rule Overview

Ensures that if a `@yields` is present that a `yield` (or `yield` with a value) is present in the function body (or that if a `@next` is present that there is a yield with a return value present).

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.
