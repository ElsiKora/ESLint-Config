# `@elsikora/jsdoc/check-types`

<Callout type="warning" emoji="⚠️">

## Rule Overview

Reports types deemed invalid (customizable and with defaults, for preventing and/or recommending replacements).

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.
