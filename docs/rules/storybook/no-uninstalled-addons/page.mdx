# `@elsikora/storybook/no-uninstalled-addons`

<Callout type="error" emoji="🚨">
	**Error.** ElsiKora reports this rule as an error in at least one preset scenario.
</Callout>

## Rule Overview

This rule identifies storybook addons that are invalid because they are either not installed or contain a typo in their name.

This page describes the rule in general first, then shows the exact ElsiKora preset configuration separately.

## Rule Metadata

| Fixable            | ❌ No     |
| Has suggestions    | ❌ No     |
| Deprecated         | ❌ No     |
